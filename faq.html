Daily Traffic and Revenue Report - Frequently Asked Questions (FAQs)

1. What is DTRR?

  • DTRR is a flash report tool used to monitor daily traffic and revenue data in MPTC expressways both domestic and foreign. Fast track and audit revenue and toll leakage instantly.

2. Who will have access to the reports/dashboards?

  • Access will be granted based on role and need. Existing users of the MPTC's Data Lake platform. Specific access levels will be defined based on job functions.

3. How to get access to the platform?

  • All the access requests will be coursed through MPTC Helpdesk. Please email helpdesk@mptc.com.ph with the subject header [1]“VPN Access Request” and [2]“Data Lake Access Request” or create a request via the SDP Portal.
  • Send a separate email thread for each access request containing the details below:
    • Full Name
    • Company
    • Department/Division
    • Immediate Superior
    • Company Email
  • Once the request is approved, the user will receive the following resources in their respective email:
    • Username and Password
    • User Manual for VPN Setup
    • URL link to Data Lake

Note: It is required to change the default password provided upon receipt of user credentials.

4. What are the data available?

  • (T – 1) Data reflected are yesterday's traffic and revenue.

5. How frequently is the data being updated?

  • Reports reflected on the dashboards are updated every 5 PM.

6. What to do if the data seems incorrect or outdated?

  • If you notice any discrepancies in the data, report it to any DSD-BDA team member.

7. How is data security and privacy handled?

  • Dashboard links can be accessed through a VPN layer (access will be also provided through the Helpdesk).

8. How to share reports and insights with the team?

  • Dashboards / Charts can be captured through an image (.PNG) or downloaded as a CSV file.

9. How does the platform improve my current workflow?

  • The new platform aims to streamline processes by providing faster access to data, more intuitive reporting tools, and automated insights, ultimately saving time and improving decision accuracy.
